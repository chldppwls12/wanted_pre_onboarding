# 🚀 wanted_pre_onboarding
프리온보딩 백엔드 코스 3차 선발 과제

*사용한 언어와 프레임워크* : Javascript - Node.js(Express) <br>
*필수 기술요건*: REST API, ORM, RDBS(Mysql) 사용

<br>

---
## 💡 서비스 개요
- 본 서비스는 기업의 채용을 위한 웹 서비스입니다.
- 회사는 채용공고를 생성하고, 이에 사용자는 지원합니다.
